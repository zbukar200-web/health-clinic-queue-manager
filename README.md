# Health Clinic Queue Manager 🏥
A Flask web application for managing patient queues in a health clinic"
## Features
- ✅ Register new patients with complete information
- ✅ View waiting list in FIFO order
- ✅ Mark patients as seen by doctor
- ✅ Track wait times automatically

## Technologies Used

- 🐍 Python 3.x
- 🌶️ Flask Web Framework
- ⏰ DateTime module for timestamping
- 📦 Collections.deque for FIFO queue
