# Health Clinic Queue Manager 🏥

A Flask web application for managing patient queues in a health clinic.

## Features

- ✅Register new patients
- ✅View waiting list in FIFO order
- ✅Mark patients as seen
- ✅Track wait times and patient statistics

## Technologies Used

-🐍Python 3
-🌶️Flask
-⏰DateTime module for timestamping
-📦Deque (Queue) for FIFO operations

## Installation

1. Clone this repository
2. Install dependencies: `pip install -r requirements.txt`
3. Run: `python app.py`
4. Open browser to: `http://127.0.0.1:5000`

## How to Use

1. Go to home page to see waiting list
2. Click "Register New Patient" to add a patient
3. Click "Mark Next Patient as Seen" to process patients
